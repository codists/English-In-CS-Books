# median

c. the value that is the middle one in a set of values arranged in order of size(中位数，中值)。示例：1.假设有一组数 1, 3, 3, 6, 7, 8, 9， 那么 median 是6。2.假设有一组数1, 2, 3, 4, 5, 6, 8, 9，那么 median 是(4+5)/2=4.5。
